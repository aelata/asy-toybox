# asy-toybox
This repository contains [Asymptote](https://github.com/vectorgraphics/asymptote) scripts for fun.

## Module
* colspc.asy - HSV, HSL, HSI, HSY, CIELAB, CIELUV, OKLab color space
  + [gallery](colspc.md)
* colpal.asy - color palette for domain coloring
  + [gallery](colpal.md)
* colpath.asy - draw path with color gradient
  + [example](colpath.md)
* domcol.asy - domain coloring of complex functions
  + [usage](domcol_usage.md)
  + [gallery](domcol_gallery.md)
* geom2.asy - miscellaneous 2D geometry functions
  + [example](geom2_example.md)
* geom3.asy - miscellaneous 3D geometry functions
  + [example](geom3_example.md)
* isoline.asy - isoline for complex argument (phase)
* nmmin.asy - Nelder-Mead minimization
  + [example](nmmin.md)
